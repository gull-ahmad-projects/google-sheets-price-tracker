# 🤖 Google Sheets Price Tracker Bot

A powerful Python automation bot that scrapes product prices from a supplier’s website and updates a Google Sheet automatically — saving time and eliminating manual work.

---

## 📌 Problem Statement

Businesses often need to update product prices in Google Sheets daily. Doing this manually is repetitive, time-consuming, and prone to errors.

---

## 💡 Solution

This bot automates the entire workflow:  
- Scrapes structured product data (ID, Name, Price) from the supplier website  
- Connects to a Google Sheet via Google Sheets API  
- Updates the sheet automatically every day  
- Logs changes and errors for tracking  

---

## ⚙️ Tech Stack

- 🐍 Python  
- 🕷️ BeautifulSoup & Requests (web scraping)  
- 📊 gspread (Google Sheets API)  
- 🔐 OAuth2 (secure authentication)  

---

## 🔥 Features

- ✅ Automated daily price updates  
- ✅ Reliable web scraping with error handling  
- ✅ Seamless Google Sheets integration  
- ✅ Secure OAuth2 authentication  
- ✅ Logs activity for monitoring  
- ✅ Saves significant manual effort
